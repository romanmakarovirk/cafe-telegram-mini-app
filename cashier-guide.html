Инструкция кассира

Telegram-бот для управления заказами

📋 Работа с заказами

Когда клиент оплачивает заказ, бот отправляет сообщение с деталями в чат кассы.

Жизненный цикл заказа:

💳 Оплачен → 🟡 Готовится → 🟢 Готов

Кнопки управления:

  • 🟡 Готовится — заказ передан на кухню
  • 🟢 Готов — заказ готов к выдаче, клиент получит уведомление
  • ⏰ 15м / 20м / 30м / 40м — сообщить клиенту примерное время готовности
Нажимайте кнопку времени сразу после принятия заказа — клиент будет знать когда подходить.

🚫 Стоп-лист

Управляйте доступностью блюд в меню.

Отключить блюдо:

/stop Плов

Бот спросит, когда блюдо вернётся (30 мин, 1 час, 2 часа, или без таймера).

Отключить всю категорию:

/stop

Без аргумента — покажет кнопки для отключения целых категорий.

Посмотреть стоп-лист:

/stoplist

Показывает все отключённые блюда с кнопками для возврата.

Если клиент оплатил заказ с позицией из стоп-листа, заказ автоматически отменяется.

💸 Возврат средств

/refund 4648

Где 4648 — номер заказа. Возврат возможен только для оплаченных заказов.

  • Клиент получит уведомление о возврате
  • Автоматически создаётся чек возврата (54-ФЗ)
  • Заказ переходит в статус «отменён»

⏸ Пауза приёма заказов

Временно приостановить приём новых заказов (например, при большой загрузке кухни).

Поставить на паузу:

/pause 30

Пауза на 30 минут. Допустимо от 1 до 480 минут.

Снять паузу:

/pause

Без аргумента — возобновляет приём заказов.

Пауза снимается автоматически по истечении указанного времени.

📊 Статистика

/stats

Статистика за сегодня: количество заказов, выручка, средний чек, топ-5 позиций.

За период:

/stats неделя
/stats месяц

⚙ Назначение чата кассой

/admin

Выполните эту команду один раз в чате, куда должны приходить заказы. Обычно это личный чат с ботом или группа кассиров.
